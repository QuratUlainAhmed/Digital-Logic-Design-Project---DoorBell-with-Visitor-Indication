# Digital-Logic-Design-Project---DoorBell-with-Visitor-Indication

REPORT: 

Introduction:
A doorbell is a simple, yet important home-automation device designed to alert the homeowner when a visitor arrives. This project focuses on creating a Doorbell with Visitor Indication, which not only rings when someone presses the switch, but also has a counter via JK flipflop that shows the number of times the doorbell has been pressed until the fourth press, after which it resets again. This system is useful for homes, small offices, shops etc.

Project Description:
2.1 Objective
The main objective of this project is to design a simple and reliable doorbell system that:
•	Alerts the user through a buzzer when the doorbell button is pressed.
•	Provides the number of times the doorbell has been rung via LEDs.
•	Ensures low cost, easy installation, and user-friendly operation

2.2 Components
•	JK Flip-Flop IC (74HC76)
•	NOT Gate IC (74HC04)
•	LEDs 
•	Resistors (10k, 1k, 220 ohm)
•	Pushbutton (ACTIVE)
•	Transistor (2N2222)
•	Buzzer
•	Battery Power Supply (9V) with 5V Convertor 
•	Breadboard
•	Jumper Wires

2.3 Working Principle
When a visitor presses the push button, the circuit is completed, allowing current to flow. This activates two things simultaneously:
•	The buzzer rings, producing an audible alert.
•	The Counter shows the number of times the bell has been pressed by LEDs, using a dual J-K flipflop.
